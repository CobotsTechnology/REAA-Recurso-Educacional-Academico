Classe string



Em C, a manipulação de uma string acontece por meio de uma sequência de caracteres disposta um caractere em cada memória como é o caso abaixo. 

                        
                            int mai()
                            {

                                char nome[6]="Lucas";
                                printf("%c", nome[0]);//L está em algum endereço de memória 
                                printf("%c", nome[1]);//u está em algum endereço de memória 
                                printf("%c", nome[2]);//c está em algum endereço de memória
                                printf("%c", nome[3]);//a está em algum endereço de memória 
                                printf("%c", nome[4]);//s está em algum endereço de memória 


                            }

Este estilo de string em C, é preciso ter cuidado, pois como a manipulação é por responsabilidade do programador, então se fizer algum acesso fora da matriz. Vai acabar invadindo outros espaço da memória ocasionando algum distúrbio no funcionamento do programa.

Mas! Em C++ a classe string se encarrega de tratar dessas manipulações sozinha, como é o caso da inclusão do valor nulo no final da stringe entre outras. Para cada string declarada ela se torna um objeto da classe string podendo assim chamar funções membros da classe. Mas também podemos tratara string como um vetor de dados da mesma forma que tratamos um array de caractere em C. Veja 

        
                            #include <string>
                            using std::string;

                            int main()
                            {

                                string str_1("Lucas");
                                cout << str_1 << endl;

                                cout << str_1[0] << endl;//Pode se manipulada caractere por caractere
                                cout << str_1[1] << endl;//idem
                                cout << str_1[2] << endl;//idem
                                cout << str_1[3] << endl;//idem
                                cout << str_1[4] << endl;//idem
                                
                            }
